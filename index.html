CatEnergy


Сайт-проект CatEnergy призначений для детального ознайомлення з нашим високоякісним кормом для котів, розробленим для забезпечення вашого пухнастого друга всіма необхідними поживними речовинами. Ресурс створено для власників котів, ветеринарів, дослідників харчування тварин, а також усіх, хто дбає про здоров'я та благополуччя своїх улюбленців.


Про життя відомих українців


Сайт-проект "Життя відомих українців",саме цей призначений для детального ознайомлення з життям і діяльністю одного з найвизначніших діячів української історії – Симона Петлюри. Ресурс розроблено для істориків, дослідників, студентів, а також усіх, хто цікавиться історією України та її боротьбою за незалежність.

PsyhoArt


Це онлайн-платформа, призначена для продажу та колекціонування унікальних цифрових творів мистецтва у форматі NFT (Non-Fungible Token). Сайт надає можливість художникам з усього світу представити свої роботи, а колекціонерам - придбати оригінальні цифрові артефакти, гарантовані технологією блокчейн.

Сайт для школи англійскої мови
